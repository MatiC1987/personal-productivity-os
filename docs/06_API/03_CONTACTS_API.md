# Contacts API

## Purpose
Provide integration layer for contact management systems (Google Contacts, CRM, address books).

## Core Operations
- Create contact
- Update contact
- Delete contact
- Sync contacts
- Search contacts
- Link contacts to tasks/events

## Data Mapping
- Contact Entity
- Email Entity
- Event Entity

## Principles
- Unified identity model
- Deduplication
- Provider abstraction
- Relationship-aware sync
