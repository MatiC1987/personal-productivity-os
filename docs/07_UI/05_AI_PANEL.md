# AI Panel

## Purpose
Central interface for interacting with system intelligence and contextual AI assistance.

## Layout
- Chat interface
- Context awareness panel
- Suggested actions feed
- Memory references viewer

## Components
- Prompt input bar
- Response stream
- Action buttons (apply / discard / modify)
- Context selector (task / project / email)

## Behavior
- Real-time streaming responses
- Deep integration with all entities
- Command execution support
- Learning from user interactions

## Interaction
- Cmd+K command entry
- Inline suggestions
- Context switching without losing state
