# Calendar View

## Purpose
Provide a visual scheduling interface for events and time management.

## Layout
- Monthly / weekly / daily switch
- Event timeline grid
- Drag & drop scheduling
- Event preview panel

## Components
- Event cards
- Time blocks
- Availability overlay
- AI scheduling suggestions

## Behavior
- Real-time sync with calendar API
- Conflict detection
- Keyboard navigation
- Smart rescheduling suggestions

## Interaction
- Drag to reschedule
- Click to expand event
- Quick create event modal
