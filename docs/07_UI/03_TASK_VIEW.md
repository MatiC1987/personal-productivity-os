# Task View

## Purpose
Detailed view for managing a single task and its full context.

## Layout
- Task header (title, status, priority)
- Description panel
- Related entities (project, emails, contacts, events)
- Activity timeline
- AI suggestions panel

## Actions
- Edit task
- Change status
- Assign / reassign
- Link documents/emails/events
- Convert to project

## Behavior
- Real-time updates
- Context-aware AI assistance
- Keyboard-first interactions
- Minimal navigation friction
