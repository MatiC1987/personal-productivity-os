# Inbox

## Purpose
Central place for incoming information (emails, tasks, notes, system events).

## Sections
- Email stream
- Quick capture notes
- Unprocessed tasks
- System notifications

## Behavior
- Auto-triage suggestions
- AI categorization
- One-click conversion to task/project
- Real-time sync

## Interaction
- Keyboard-first processing
- Swipe/command actions
- Minimal friction processing flow
